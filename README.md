# Unexplained Phenomena Research Index

A comprehensive, scientific analysis of unexplained natural phenomena from across the world.

**Live Site:** https://ckorhonen.github.io/unexplained-phenomena/ *(will be active after deployment)*

## Overview

This project presents evidence-based research on phenomena ranging from well-studied but incompletely understood natural events (ball lightning, Tunguska event) to persistent mysteries with limited reproducible evidence (spontaneous human combustion) to phenomena now largely explained or debunked (Bermuda Triangle).

### Evaluation Criteria

Each phenomenon is evaluated based on:
- **Current scientific understanding**
- **Competing theories and hypotheses**  
- **Evidence quality and methodological gaps**
- **Research feasibility and potential impact**

## Methodology

Research compiled using:
- Peer-reviewed scientific literature
- Field study reports
- Archaeological excavation data
- Spectroscopic and instrumental measurements
- Historical documentation
- Expert consensus assessment

**Evidence Quality Standards:**
- **HIGH:** Documented observations, spectral data, laboratory reproduction, physical remains
- **MODERATE:** Multiple credible reports, physical traces, inconsistent data
- **LOW:** Anecdotal only, no verified cases, contradictory accounts
